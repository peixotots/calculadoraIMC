# Programação para Dispositivos Móveis - Trabalho 2

## Calculadora de IMC

### Descrição
Aplicativo Android para calcular o IMC (Índice de Massa Corporal), desenvolvido em Kotlin com Jetpack Compose.

### Funcionalidades
1. Cálculo do IMC a partir de peso e altura.
2. Classificação do resultado (baixo peso, normal, sobrepeso, obesidade).
3. UI responsiva com Jetpack Compose.
4. Validações básicas de entrada.

### Estrutura principal
1. `app/src/main/java` - código-fonte Kotlin
2. `README.md` - documentação

### Como executar (Windows)
1. Abra o projeto em Android Studio.
2. Conecte um dispositivo ou inicie um emulador.
3. No terminal do projeto:
   - PowerShell: `.\gradlew assembleDebug` e depois `.\gradlew installDebug`
   - CMD: `gradlew.bat assembleDebug` e depois `gradlew.bat installDebug`
4. Alternativamente, use o botão Run do Android Studio.
